## [0.1.0] — 2026-04-17

### Added

**Templates (10)**

| # | Name | Pattern | Safety |
|---|------|---------|--------|
| 1 | `hello-grok` | single-agent | standard |
| 2 | `reply-engagement-bot` | multi-step | strict |
| 3 | `trend-to-thread` | multi-step | strict |
| 4 | `research-swarm` | swarm | standard |
| 5 | `code-reviewer` | multi-step | strict |
| 6 | `thread-ghostwriter` | multi-step | strict |
| 7 | `personal-knowledge` | multi-step | standard |
| 8 | `scientific-discovery` | swarm | standard |
| 9 | `voice-agent-x` | multi-step | strict |
| 10 | `live-event-commentator` | multi-step | strict |

All 10 templates are **Grok-Native Certified** (see README for criteria).

**Infrastructure**

- CI workflow (`validate-templates.yml`) running structural validation,
  security scan, mock tool execution, and YAML lint on every PR.
- In-repo validator scripts (`validate_template.py`, `validate_registry.py`,
  `scan_template.py`, `mock_run_template.py`) — no external CLI dependency.
- `grok_install_stub` — installable stub runtime package for CI use.
- SVG poster cards for every template (`docs/posters/`), generated by
  `scripts/gen_poster.py` from `featured-agents.json` and agent YAML.
- Contributor tooling: PR template, issue templates, `CONTRIBUTING.md`,
  `docs/submitting-your-own.md`, `docs/template-anatomy.md`.
